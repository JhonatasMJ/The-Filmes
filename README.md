## Tecnologias Utilizadas 💻

- Tailwind
- Firebase
- Zod

## Como Usar❓

    1. Clone este repositório em sua máquina local.
    2. Execute o comando "npm install" no terminal
    3. Digite o comando "npm run dev" para rodar o projeto
